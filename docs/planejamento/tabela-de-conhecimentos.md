# Tabela de Conhecimentos

## Quadro de Conhecimento da Equipe

O quadro de conhecimento é um artefato essencial para avaliar o nível de conhecimento de cada membro da equipe em diversas tecnologias. Essa avaliação permite identificar áreas de melhoria, orienta a escolha de tecnologias a serem utilizadas no projeto, auxilia na distribuição eficiente de tarefas e permite o planejamento de eventuais treinamentos.

## Tecnologias Selecionadas

Na construção do quadro de conhecimentos, foram selecionadas as seguintes tecnologias e competências técnicas que a equipe julgou importante avaliar:

* Python
* TypeScript
* Java
* C/C++
* Node/Bun
* React
* Vue
* API Rest
* Docker
* CI/CD
* Git
* Testes (JUnit/Jest/pytest)
* Nginx/Traefik
* Deploy
* VPS

## Classificação do Nível de Conhecimento

Após a definição das tecnologias, cada membro da equipe preencheu uma linha da tabela com o respectivo nível que representa sua proficiência nas ferramentas listadas anteriormente. Os níveis disponíveis foram:

* **Nenhum:** Nunca usei na prática; preciso aprender do zero.		
* **Básico:** Faço tarefas simples, mas dependo de tutoriais ou ajuda.
* **Intermediário:** Tenho autonomia para trabalhar sozinho e resolver o dia a dia.
* **Avançado:** Domino a tecnologia, resolvo problemas complexos e consigo ensinar.

Depois que todos os membros realizaram o preenchimento, foi feita uma análise considerando o domínio da equipe em cada uma das tecnologias por meio de uma soma ponderada, baseando-se nos níveis estabelecidos.

Essa análise ajuda a criar um panorama claro das competências técnicas do grupo, facilitando a identificação de especialistas em cada área, a definição da arquitetura do produto e o mapeamento de membros que possam necessitar de mais suporte.
