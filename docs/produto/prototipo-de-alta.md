# Protótipo de Alta Fidelidade

Um protótipo de alta fidelidade é uma simulação refinada do produto final. Onde detalhes do design visual e conteúdo real mostram a aparência e a funcionalidade do produto final. A prototipagem de alta fidelidade desempenha um papel fundamental nas etapas finais do processo de design, desde os testes de usabilidade até a entrega aos desenvolvedores.

Neste documento serão expostas as telas prototipadas para o sistema web RetinaScan. Caso seja necessário a criação de novas telas, elas seguirão os mesmos elementos e componentes. Todo o protótipo foi produzido através do editor gráfico Figma.
